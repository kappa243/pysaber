# pySaber

Python implementation of [Saber MLWR-based KEM](https://www.esat.kuleuven.be/cosic/pqcrypto/saber/).

## How to run locally

1. Clone the repository and enter project directory.
2. Install poetry:
```bash
pip install poetry
```
3. Install dependencies:
```bash
poetry install
```
4. Run main cli:
```bash
poetry run main
```
